Color Seperator for Risograph Printmaking

Figure 1: Result of color seperation with seafoam, flourescent pink, blue, and yellow riso inks

A risograph machine is a type of digital printer used by artists. It uses a stencil-like process with an ink roller to print one layer of color at a time. This project was motivated because the software that art students use to seperate images into layers of different ink colors that the Risograph printer has available is only useable on mac.

I wrote a Matlab script to separate images into a custom palette of pantone colors. Least squares estmation was used to find a linear combination of the given colors' RGB values that best approximates the image.

Figure 2 and 3: Original image (left) and overlayed seperated image (right)

The image is not always perfectly replicated by this process unless the RBG values of the colors selected are linearly independant. The image above is seperated into blue, fluorescent pink, and yellow. There are also ink colors that are available for the risograph (riso) printer that are not in the RGB color gamut. In these cases the closest RGB value is used instead. Different riso inks also have different opacities which leads to unexpected outcomes.
